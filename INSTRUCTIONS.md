# Custom Icons Instructions

## How to add custom icons:

1. Save your icon files in the same folder as your HTML files (c:\Users\mrhau\Documents\html\kauri)
2. Make sure to name your icon files correctly as listed below:

## Required icon files:

| Icon Name          | File Name          | Recommended Size |
|---------------------|--------------------|-----------------|
| Kauri Logo          | `kauri-logo.png`   | 80x80px         |
| Roblox Icon         | `roblox-icon.png`  | 64x64px         |
| Discord Icon        | `discord-icon.png` | 64x64px         |
| TikTok Icon         | `tiktok-icon.png`  | 64x64px         |
| X (Twitter) Icon    | `x-icon.png`       | 64x64px         |

## Tips:
- Use PNG files with transparent backgrounds for best results
- You can use JPG or GIF files too if you want
- If you don't add an icon, the image will not show up (just the text will be there)
- You can change the icon sizes in the CSS if you want!

## To edit link URLs:
Just open each HTML file and replace the `#` in the `<a href="#">` with your actual link!
